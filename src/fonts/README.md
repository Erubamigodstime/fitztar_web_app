# Fonts Directory

## Creato Display Font Files

Please add the following Creato Display font files to this directory:

1. **CreatoDisplay-Bold.woff2** (Weight: 700)
2. **CreatoDisplay-Bold.woff**
3. **CreatoDisplay-Regular.woff2** (Weight: 400)
4. **CreatoDisplay-Regular.woff**

## Font Specifications

- **Bold**: Weight 700, Size 48px (headings)
- **Regular**: Weight 400, Size 16px (body text)

If you don't have the font files, you can:
1. Purchase them from the official source
2. Use a similar alternative like "Poppins" or "Montserrat" from Google Fonts
